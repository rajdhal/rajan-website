# Product Overview

This is a personal portfolio website for Rajan Dhaliwal, a software engineer. The site showcases:

- Professional profile and introduction
- Technical skills across languages, frameworks, databases, and tools
- Project portfolio with descriptions and links

The portfolio is designed to be modern, responsive, and easy to navigate, serving as a professional web presence for career opportunities and networking.

**Live Site**: https://rajdhal.github.io/portfolio-website/

**Target Audience**: Recruiters, potential employers, collaborators, and professional contacts
